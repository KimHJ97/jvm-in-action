## 로그가 있어서 가능한 일들

 - 운영하다가 장애가 발생했는지 로그를 통해 체크
 - 장애 발생시 문제 해결을 위한 힌트 제공
 - CS(고객문의)가 들어와 서비스에서 처리된 데이터 확인
 - 개인정보를 다루는 서비스나 금융 관련 서비스에서는 법적으로 로그를 남겨야 하는 경우도 있음
