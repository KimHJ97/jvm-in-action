## 강의 소개

 - 장애 없는 서비스를 만들기 위한 CircuitBreaker의 개념
 - Resilience4j를 통해 CircuitBreaker를 빠르게 적용

Resilience는 회복 탄력성(장애 내성)이라는 뜻으로 어떤 충격을 겪었을 때, 충격으로부터 빠르게 피해를 극복하는 성질을 말한다.  
트래픽이 집중되는 서비스에 서킷 브레이커를 걸어 트래픽을 효과적으로 제어하고, 서비스를 안정적으로 유지하는 방법을 배운다.
